Ready to send Email

Include the libraries

Iclude the folowing files in your head section: stylesheet ReadyToSendMailer.css, jquery jquery-1.8.3.min.js, Ready to Send Mailer ReadyToSendMailer.js

			
    
			
  1. 
<link href="css/ReadyToSendMailer.css" rel="stylesheet" type="text/css" />
<script src="js/jquery-1.8.3.min.js" type="text/javascript"></script>
<script src="js/ReadyToSendMailer.js" type="text/javascript"></script>
    
			
    2.

Add a DIV element

That DIV will contains the contact form, Ready to Send Mailer will build the necesary elements for you

			
    
   
			
  1. 
<body>
	<div id="mail" class="mail" style="width: 400px"></div>
</body>
			
    2.

Call Ready to Send Mailer

    

  1. 
    <script type="text/javascript">
        $(document).ready(function () {
            $('.mail').ReadyToSendMailer({
                template: 'medium',
                url: 'php/email_medium.php',
                base: 'black',
                theme: 'pink',
				myLocation: 'California City, California',
		css3Enter: 'fadeInRightBig',
		css3Exit: 'fadeOutRightBig',
                social: {
                    facebook: "http://facebook.com",
                    twitter: "http://twitter.com",
                    youtube: "http://youtube.com"
                }
            });
        });
	</script>

Ready To Send Mailer has a set of options to customize its appearance and functionality

Option Description Default Values
template Choose the contact form layout size medium small, medium, large
width Width of the contact form 400
url Address of the php file that sends the email php/email_medium.php
base Base color of the form black black, white
theme Elements color theme sky sky, orange, dark, gold, silver, pink, grass, red
css3Enter Name of the enter animation fadeInUpBig see the list in this page
css3Exit Name of the exit animation fadeOutDownBig see the list in this page
modal choose whether the form appears in a modal window true true, false
drag choose if you want to drag a item to show the contact form true true, false
myLocation input your location if you want to show the full screen map false false, your location
social Show the social links { facebook: "http://facebook.com", twitter: "http://twitter.com" } false, links
